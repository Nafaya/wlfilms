#Установка
```bash
git clone git@github.com:Nafaya/wlfilms.git
cd wlfilms
npm install
```
##Запуск
```bash
npm start
```
##Продакшин
Сборка
```bash
npm run build
```
Запуск
```bash
Windows:
set NODE_ENV=production&&npm start
Ubuntu/Mac:
export NODE_ENV=testing&&npm test
```
